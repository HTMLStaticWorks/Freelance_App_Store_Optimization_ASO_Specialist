# Freelance ASO Specialist - Premium Portfolio Website

A complete, responsive, and high-end portfolio website for an App Store Optimization (ASO) specialist. Built with Vanilla HTML5, CSS3, and JavaScript.

## Features
- **13+ Premium Pages**: Includes two homepage variants, client portal, dashboard, and more.
- **Modern Design**: Glassmorphism, bento grids, and gradient aesthetics.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop (320px+).
- **Dark Mode & RTL Support**: Integrated theme and language direction switching.
- **Interactive UI**: Animated counters, floating elements, and scroll-reveal effects.
- **SEO Ready**: Semantic HTML, meta tags, sitemap, and robots.txt included.

## Folder Structure
- `/assets`: CSS, JS, and image assets.
- `/index.html`: Home Page 1 (Main).
- `/home-2.html`: Home Page 2 (Premium Variant).
- `/dashboard.html`: Client Analytics Portal.
- `/login.html` & `/register.html`: Authentication pages.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Outfit)
